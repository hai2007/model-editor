# Mtl文件描述

也就是Obj文件的材质文件。

```python
# Blender MTL File: 'None'
# Material Count: 1

newmtl Material #使用newmtl定义了一个新的材质，可以被cube.obj文件来引用我这里定义的材质纹理信息
Ns 323.999994 #使用Ns 来指定高光色的权重
Ka 1.000000 1.000000 1.000000 # ka,kd,ks分别定义了表面的环境色，漫射色和高光色，使用RGB格式, 每一个分量的值都是介于[0.0, 1.0]这个区间的
Kd 0.800000 0.800000 0.800000
Ks 0.500000 0.500000 0.500000
Ke 0.000000 0.000000 0.000000
Ni 1.450000 #Ni指定了表面光学密度
d 1.000000 #d 指定了透明度
illum 2 #illum 指定了光照模型
```
